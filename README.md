This is my final or capstone project for boot.dev. As the name implies it's a monster tamer game.

A very simple monster tamer.

It's mostly just the basic concept of a monster tamer game.
It has a basic battle, a few monsters and a couple moves, there are some items as proof of concepts.

Continue loads the save state and new game erases your save data. If you do press continue without
a previous save it will start a new save. 

you nagivate by using the arrow keys and enter/return to select an menu option. You can go back with
with backspace or escape.

Current Monsters:
testmon, moxmon, timon, huntmon, steamon

Current Moves:
tackle, fireball, firesteam, heal, vine wrap

Current Items:
Catch items: Catcher
Healing items: Potion
Training items: weights, treadmill, book, jump rope, heart stone, punching bag

Monsters have the following stats:
HP, Energy, Courage, Power, Wisdom, Speed, and Elements.

Some moves have elements as well that boost damage or add additional effects.
The training items allow you to give bonus stats to your monster, but only to a 
max of 10 to a single stat and 25 additional poins total. Each item adds 5 points.
